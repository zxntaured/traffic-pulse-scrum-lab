# TrafficPulse

**TrafficPulse** – это консольное приложение для анализа ежедневных маршрутов. Оно помогает пользователю выбрать оптимальное время для поездки, визуализируя гипотетическую или реальную загруженность на пути.

Проект использует Python и фейковые/открытые API для симуляции данных. Цель – не создать продакшен-решение, а отработать процесс командной разработки по методологии Scrum с использованием GitHub Issues.

Ключевые возможности, которые будут реализованы: парсинг входных данных, расчет маршрута, генерация "пробок", ASCII-визуализация, рекомендательная система и экспорт отчета.
